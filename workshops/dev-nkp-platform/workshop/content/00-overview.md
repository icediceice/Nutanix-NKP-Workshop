## What We're Doing

NKP (Nutanix Kubernetes Platform) is more than Kubernetes — it ships a curated set of platform
services that cover the common needs of development teams. This workshop introduces those
services from a developer perspective: what they are, how to access them, and how to use them
in your application workflow.

## Platform Services Covered

| Service | Purpose |
|---------|---------|
| Harbor | Private container registry with vulnerability scanning |
| Grafana + Prometheus | Metrics and dashboards |
| OpenSearch + Dashboards | Log aggregation and search |
| Application Catalog | One-click deploy of databases, middleware, tools |
| FluxCD | GitOps continuous delivery |
| Istio | Service mesh: traffic management, mTLS, observability |
