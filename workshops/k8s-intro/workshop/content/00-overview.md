## What We're Doing

Welcome to the Kubernetes Introduction workshop. In this session you will get hands-on experience
with the core concepts that underpin every Kubernetes deployment. By the end you will be able to
navigate a cluster, inspect workloads, and run your first container — all from the command line.

## What You Will Learn

- How to connect to a Kubernetes cluster and verify your context
- The anatomy of a Kubernetes node and how the control plane works
- How to run, inspect, and delete Pods
- How to read `kubectl` output and understand what it means

## Exercises

| # | Topic | Time |
|---|-------|------|
| 01 | Connect to the cluster | 10 min |
| 02 | Explore nodes | 15 min |
| 03 | Run your first Pod | 20 min |

## Prerequisites

- Access to the workshop cluster (credentials provided by your facilitator)
- `kubectl` installed on your workstation or use the in-browser terminal

## A Note on the Terminal

Every code block marked with `terminal:execute` will run automatically in the embedded terminal.
You can also type commands yourself — experimentation is encouraged.

```terminal:execute
command: echo "Ready to explore Kubernetes!"
```

**Observe:** You should see the message printed to the terminal. This confirms your session is live.
