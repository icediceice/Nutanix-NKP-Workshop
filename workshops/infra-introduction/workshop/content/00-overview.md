## What We're Doing

The infrastructure track begins here. Before managing Kubernetes clusters you need to understand
what runs beneath them: the Nutanix Hyperconverged Infrastructure platform and the Cluster API
framework that NKP uses to provision clusters declaratively.

## What You Will Learn

- How Nutanix AHV and AOS form the compute, storage, and network substrate for NKP clusters
- What Cluster API (CAPI) is and why it matters for multi-cluster management
- How NKP uses CAPI to provision, upgrade, and scale Kubernetes clusters on Nutanix
- The lifecycle of an NKP cluster from API request to running nodes

## Prerequisites

- Familiarity with virtualisation concepts (VMs, hypervisors, storage)
- Completion of the Kubernetes Introduction workshop recommended but not required
- NKP management cluster access provided by your facilitator
