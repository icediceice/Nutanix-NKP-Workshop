## What We're Doing

You provisioned a cluster in the Infrastructure Introduction workshop. Now you will manage it
at scale using NKP's enterprise features: Kommander workspaces for multi-tenancy, role-based
access policies, kubeconfig federation, storage class configuration, and backup/DR with Velero.

## What You Will Learn

- How Kommander workspaces organise clusters, teams, and policies
- How to configure access policies and propagate RBAC across clusters
- How to obtain and distribute kubeconfig files for managed clusters
- How Nutanix StorageClasses are configured for different workload profiles
- How Velero backs up workloads and enables cross-cluster disaster recovery
